# Redu-o-dos-Custos-em-Farm-cias-com-AWS---Bootcamp-Santander-
Este relatório apresenta o processo de implementação de ferramentas na empresa Abstergo Industries (nome fictício). O objetivo do projeto foi elencar 3 serviços AWS, com a finalidade de realizar diminuição de custos imediatos e modernizar a plataforma da farmácia virtual.
